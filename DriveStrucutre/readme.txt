steps to run this application to get the json comparision are as follows : 

==> open command prompt 

1) check whether angular is installed in local machine or not 
   ---> if not then open cmd and write following command to install angular 
  		command :  npm install -g @angular/cli
  
2) once step 1 is done ,then run following command on cmd :
   ---> npm install -g json-diff
   
3) once step 1 and 2 is done then go to Constants.java file and provide source(line no. 11)
   and destination(line no. 13) file path 
   
4) once all above steps are done then go to Main.java and run the application
